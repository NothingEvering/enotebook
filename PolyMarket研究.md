## 一、平台运转机制

### 核心概念

Polymarket 让用户对真实世界事件的结果进行交易，每个市场都是一个二元 **是/否** 问题。

- **份额价格 = 市场隐含概率**：$0.30 的份额意味着市场认为该事件有 30% 概率发生
- **结算规则**：
  - 事件发生 → YES 份额每份兑换 **$1.00 USDC**，NO 份额归零
  - 事件未发生 → NO 份额每份兑换 **$1.00 USDC**，YES 份额归零
  - 1 份 YES + 1 份 NO 永远等于 **$1.00**（无套利约束）
- **可提前退出**：不必持有到结算，可随时在市场中买卖份额

### 技术架构

| 组件 | 说明 |
|------|------|
| 区块链 | Polygon（Gas 费极低，速度快） |
| 结算货币 | USDC.e（Polygon 上的 USDC） |
| 订单簿模型 | 中央限价订单簿（CLOB），2022 年底从 AMM 迁移 |
| 撮合方式 | 链下撮合（低延迟）+ 链上结算（去信任化） |
| 性能 | 平均延迟 45ms，峰值 3,200 笔订单/秒 |
| 结果认定 | 预言机 + 市场诚信委员会（Market Integrity Committee） |
| 智能合约 | 已通过 ChainSecurity 安全审计 |

### 账户注册

- 连接 Web3 钱包（MetaMask、Coinbase Wallet 等）即可开始交易
- **全球用户无需 KYC**（美国用户除外，需通过受监管通道）
- 存入 USDC 后立即可交易

## 二、盈利策略

> **重要数据**：根据对 9500 万笔链上交易的分析，**80% 的参与者亏损**，盈利超过 $1000 的钱包地址仅占 **0.51%**。这是一个竞争极为激烈的市场。

### 策略一：信息套利（最具潜力）

利用自身信息优势或分析能力，在市场定价错误时入场。

- **方法**：用 AI 模型、新闻聚合或领域专业知识估算概率，与市场价格对比
- **示例**：多个 AI 模型给出 65-71% 的概率，而市场仅定价 54%，存在 14 个百分点的价差
- **适合人群**：研究人员、记者、领域专家、有快速信息获取能力的交易者

### 策略二：跨平台套利

利用 Polymarket 与竞争对手（Kalshi、PredictIt、Manifold）之间的价格差异。

- **示例**：某事件在 Polymarket 的 YES 价为 $0.45，在 Kalshi 的 NO 价为 $0.48 → 两边各买一份合计 $0.93，结算收 $1.00，**无风险收益 7.5%**
- **适合人群**：系统化交易者、算法交易者

### 策略三：做市商（流动性提供）

同时挂出买单和卖单，赚取买卖价差。

- **预期收益**：历史数据显示月均收益 1-3%，胜率 78-85%
- **激励机制**：每日分发 Maker 返佣（Taker 手续费的 20-50%，按类别不同）
- **要求**：需通过 CLOB API 进行技术接入，适合量化交易者和开发者
- **适合人群**：开发者、量化交易者、自动化系统

### 策略四：高概率"类固收"策略

寻找某一方向几乎确定但市场还未充分定价（如未到 0.95-0.99）的市场，买入并持有至结算。

- 回报上限低，但所需精力最少
- **适合人群**：新手、被动交易者

### 策略五：垂直领域专精

在某一细分领域（地缘政治、体育、加密）建立信息优势。

- 用 PolyTrack 等工具跟踪顶级交易者的持仓
- 系统性发现自己擅长领域内的定价错误
- **核心要求**：严格的仓位管理，单个市场不超过可承受损失

### 策略六：速度/新闻交易

在重大突发事件后，比市场更快反应并建仓。

- 需要实时新闻源和快速执行能力
- **注意**：这个方向竞争非常激烈，算法机器人主导

## 三、入金方式

Polymarket 的所有资产以 **Polygon 链上的 USDC.e** 形式存储，所有入金均会自动转换。

### 方式一：加密货币直接转账（首选，费用最低）

| 项目 | 说明 |
|------|------|
| 支持网络 | 以太坊、Polygon、Arbitrum、Base、Optimism、BNB Smart Chain、Solana、Bitcoin、Tron 等 **14 条链** |
| 支持代币 | USDC、ETH、BTC、SOL、BNB、USDT、DAI 等，自动兑换为 USDC.e |
| 到账速度 | 通常几分钟内 |
| 最低金额 | L2 链 $2，以太坊主网 $7，BTC/Tron $9 |
| Polymarket 收费 | **免费**（仅需支付链上 Gas，Polygon 上极低） |

**推荐流程**：交易所购买 USDC → 选择 Polygon 网络 → 提币到 Polymarket 钱包地址

### 方式二：法币直接购买（在平台内集成）

| 支付方式 | 服务商 | 手续费 |
|----------|--------|--------|
| 信用卡/借记卡 | MoonPay、Stripe | 3% – 4.5% |
| 银行转账（ACH/SEPA） | 集成服务 | ~1% 或少量固定费用 |
| Apple Pay / Google Pay | 支持地区可用 | 同卡费率 |
| Robinhood Connect | 美国用户专属 | 视账户情况 |

> **建议**：如果已有加密货币账户，优先使用方式一。法币入金的 3-4.5% 手续费会显著侵蚀利润。

## 四、出金方式

### 出金流程

Polymarket **不支持直接提现到银行账户**，需经过加密货币中间步骤：

```
Polymarket 钱包
    ↓ 发送 USDC 到交易所地址（近乎即时，几秒到几分钟）
中心化交易所（如 Coinbase、OKX、币安）
    ↓ USDC 兑换为法币（USD/CNY/EUR 等）
    ↓ 提现到银行账户
银行账户
```

### 费用明细

| 环节 | 费用 |
|------|------|
| Polymarket 提款费 | **$0** |
| Polygon 链 Gas 费 | 不足 $0.01（可忽略） |
| USDC 兑换滑点 | ~0.08% |
| 交易所法币提现 | 各平台不同，通常 0-1% |

## 五、费用结构

### 交易手续费（2026-03-30 起生效）

Polymarket 对几乎所有类别的市场开始征收 **Taker 手续费**，费率随价格变化（价格越接近 $0.50 费率越高，越接近 $0.00 或 $1.00 费率越低）。

| 市场类别 | 在 $0.50 时的峰值有效费率 |
|----------|--------------------------|
| 加密货币 | ~1.80% |
| 舆论/经济 | ~1.50% |
| 政治、金融、体育、文化、天气 | 0.75% – 1.25% |
| 地缘政治/世界事件 | **0%（免手续费）** |

- **费率公式**：`fee = C × p × feeRate × (p × (1-p))^exponent`
- **Maker 返佣**：每日将收取的 Taker 手续费的 20-50% 返还给流动性提供者

### 美国用户专用通道（Polymarket US / QCEX）

| 类型 | 费率 |
|------|------|
| Taker 费 | 0.30% |
| Maker 返佣 | -0.20% |

### 无费用项目

- Polymarket 平台侧入金和出金均不收费
- 亏损的交易不收费（费用只在你主动买卖时产生）

## 六、风险与注意事项

### 安全风险

| 风险类型 | 说明 |
|----------|------|
| 第三方认证漏洞 | 2025 年底，邮箱登录使用的第三方认证服务（Magic Labs）被攻击，导致账户被盗，平台已修复但核心协议未受影响 |
| 钓鱼攻击 | 市场评论区出现钓鱼链接，指向伪造的 Polymarket 页面，已有数十万美元损失 |
| 恶意工具 | 伪装成 Polymarket 库的 npm 包被用于窃取钱包私钥 |

**防护建议**：
- 使用硬件钱包，永不泄露助记词
- 不点击市场评论中的链接
- 只从官方渠道下载工具

### 市场/财务风险

- **80% 的交易者亏损**，市场中有大量专业机构和算法机器人
- **流动性风险**：小市场买卖价差大，大额交易会产生滑点
- **市场操纵**：历史上存在有据可查的操纵案例（如 2024 年 $8500 万特朗普押注引发调查）
- **结算争议**：即使预测正确，结果认定机制也可能产生争议

### 监管/法律风险

- 美国各州监管不统一，内华达、田纳西、马萨诸塞州有执法行动
- 全球多个国家和地区明确禁止访问

### 道德争议

- 2026 年 3 月，有用户威胁以色列记者，要其改变关于伊朗导弹袭击的报道，以影响相关市场价格
- 预测市场的激励机制可能产生操纵现实事件的动机

## 七、地区限制

### 美国

- **2022-2025 年封锁**：2022 年 1 月因违规运营被 CFTC 罚款 $140 万
- **2025 年 12 月 2 日重新开放**：通过收购持牌衍生品交易所 QCEX（价值 $1.12 亿）并获得 CFTC 批准后重返美国市场
- **州级风险持续**：内华达、田纳西等州仍有执法行动（截至 2026 年初）

### 完全封锁的国家/地区（共 33+ 个，部分列举）

- 法国（被 ANJ 赌博监管机构封禁）
- 比利时（被 BGC 封禁）
- 瑞士、罗马尼亚、葡萄牙、澳大利亚
- 乌克兰（KRAIL 以"对军事结果下注"为由封禁）

### 仅可查看、不可开仓

- 新加坡、波兰、泰国、台湾
- 加拿大安大略省（禁止存款和交易）

### 全球覆盖

- 在约 **180 个国家**无限制运营
- 全球用户无 KYC 要求（仅需连接钱包）
